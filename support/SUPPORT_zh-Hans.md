## 功能介绍

### ✨ 核心功能

#### 1. 时间轴故事记录
- **功能描述**：以时间线形式展示您的所有故事，直观查看生活轨迹
- **支持内容**：
  - 文字记录（支持富文本格式）
  - 图片（多图支持）
  - 视频（自动生成缩略图）
  - 位置信息（可选）
- **操作方式**：
  - 点击右上角 "+" 按钮创建新故事
  - 上下滑动浏览历史故事
  - 点击卡片查看完整内容

#### 2. AI 文本润色
- **功能描述**：使用 AI 智能优化您的文字表达
- **使用场景**：
  - 润色日记文字
  - 优化语言表达
  - 调整文章风格
- **如何使用**：
  - 在故事编辑页面选择文本
  - 点击工具栏的 AI 润色按钮
  - 查看并选择润色结果

#### 3. 三级分类管理
- **功能描述**：灵活的分类体系，帮助您整理故事
- **分类层级**：
  - 一级分类：大类（如：旅行、工作、生活）
  - 二级分类：中类（如：旅行 > 国内游）
  - 三级分类：细类（如：旅行 > 国内游 > 杭州之旅）
- **自定义选项**：
  - 自定义分类名称
  - 选择图标（SF Symbols 或自定义图标）
  - 设置显示颜色
  - 设置显示模式（卡片/列表）

#### 4. 多媒体支持
- **图片功能**：
  - 支持从相册选择
  - 支持相机拍摄
  - 支持多图上传
  - 支持图片裁剪
- **视频功能**：
  - 支持短视频记录
  - 自动生成缩略图
  - 支持视频预览播放

#### 5. 位置记录
- **功能描述**：记录故事发生的地点
- **支持功能**：
  - 自动获取当前位置
  - 手动搜索地点
  - 位置搜索历史
  - 地图预览（预留）
- **隐私保护**：
  - 位置信息仅本地存储
  - 不上传到任何服务器
  - 可随时删除位置信息

#### 6. 搜索功能
- **搜索范围**：
  - 故事标题
  - 故事内容
  - 分类名称
  - 位置信息
- **搜索功能**：
  - 关键词高亮显示
  - 搜索历史记录
  - 快速定位结果

#### 7. 设置与个性化
- **语言设置**：
  - 简体中文
  - English
  - 实时切换，无需重启
- **主题设置**：
  - 7 种主题可选
  - 支持浅色/深色模式
  - 实时预览效果
- **字体设置**：
  - 4 种字体大小（小、标准、大、超大）
  - 实时预览调整效果

#### 8. 数据同步
- **设备间迁移**：
  - 点对点数据传输
  - 无需联网服务器
  - 端到端加密传输
  - PIN 码验证安全性
- **数据备份**：
  - 创建本地备份
  - 快速数据恢复
  - 保留完整数据结构

## 常见问题

### Q1: 如何创建新故事？
**A:** 
1. 在时间轴页面，点击右上角的 "+" 按钮
2. 输入故事标题和内容
3. （可选）添加图片、视频或位置信息
4. （可选）选择分类
5. 点击"保存"完成创建

### Q2: 故事可以添加几张图片？
**A:** 9张图数量限制，您可以添加任意数量的图片。但为了获得最佳浏览体验，建议单个故事不超过 9 张图片。

### Q3: 如何使用 AI 文本润色功能？
**A:** 
1. 在故事编辑页面，输入或选择需要润色的文本
2. 点击工具栏的 "AI 润色" 按钮
3. 等待 AI 处理（通常需要 2-5 秒）
4. 查看润色结果，选择满意的版本
5. 点击"应用"将润色后的文本替换到编辑器中

**注意**：使用 AI 功能需要联网，您的文本会发送到阿里云通义千问 API 进行处理。

### Q4: 位置信息是必须的吗？
**A:** 不是必须的。位置信息是可选功能，您可以根据需要选择是否添加。如果不需要，可以在系统设置中关闭位置权限。

### Q5: 如何创建和管理分类？
**A:** 
1. 点击底部导航栏的"分类"标签
2. 点击右上角的 "+" 按钮创建新分类
3. 填写分类名称，选择层级和父分类
4. 选择图标和颜色
5. 点击"保存"完成创建

**编辑分类**：在分类列表中，向左滑动分类卡片，选择"编辑"或"删除"。

### Q6: 删除分类会发生什么？
**A:** 删除分类时会显示确认对话框，告知您将会删除的内容：
- 该分类本身
- 该分类的所有子分类
- 该分类下的所有故事
- 相关的所有媒体文件

**此操作不可恢复**，请谨慎操作。建议删除前先导出重要数据。

### Q7: 如何切换语言？
**A:** 
1. 进入"设置"页面
2. 点击"语言"选项
3. 选择您想要的语言（简体中文 或 English）
4. 点击"完成"
5. 应用界面会立即切换到所选语言

### Q8: 如何更换主题？
**A:** 
1. 进入"设置"页面
2. 点击"主题"选项
3. 浏览并选择喜欢的主题
4. 主题会实时应用到所有页面

可选主题包括：经典、海洋、晚霞、夜空、森林、薰衣草、暗夜。

### Q9: 如何调整字体大小？
**A:** 
1. 进入"设置"页面
2. 点击"字体"选项
3. 拖动滑块或点击预设选项（小、标准、大、超大）
4. 实时预览效果
5. 点击"完成"保存

### Q10: 如何搜索故事？
**A:** 
1. 在时间轴页面顶部，点击搜索栏
2. 输入关键词（可以搜索标题、内容、位置、分类）
3. 查看搜索结果，关键词会高亮显示
4. 点击结果查看完整故事

**搜索历史**：搜索栏会显示您最近的搜索记录，点击即可快速搜索。

### Q11: 数据存储在哪里？
**A:** 
- **所有数据都存储在您的设备本地**
- 文字内容存储在 Core Data 数据库中
- 图片和视频使用 AES-256-GCM 加密后存储在应用沙盒目录
- 加密密钥存储在 iOS 系统钥匙串中
- **我们不会将您的数据上传到任何服务器**

### Q12: 如何迁移数据到新手机？
**A:** 使用内置的"数据同步"功能：

**旧手机（发送端）**：
1. 进入"设置" > "数据同步"
2. 点击"创建本地同步备份"
3. 点击"开始连接"搜索接收端
4. 输入新手机显示的 PIN 码
5. 点击"发送 PIN 验证并传输"

**新手机（接收端）**：
1. 进入"设置" > "数据同步"
2. 点击"等待接收"
3. 记下显示的 6 位 PIN 码
4. 等待旧手机连接并传输
5. 传输完成后点击"恢复数据"

**注意**：两台设备需要在同一 WiFi 网络下，或开启蓝牙。

### Q13: 如何清理缓存？
**A:** 
1. 进入"设置"页面
2. 点击"缓存清理"选项
3. 确认清理操作
4. 系统会清理临时文件和压缩包

**说明**：缓存清理不会影响您的故事和媒体数据，只会删除临时文件。

### Q14: 应用崩溃或无响应怎么办？
**A:** 
1. **强制退出应用**：双击 Home 键（或从底部上滑），找到应用并向上滑动关闭
2. **重启设备**：长按电源键，滑动关机，然后重新开机
3. **检查存储空间**：确保设备有足够的可用存储空间（建议至少保留 1GB）
4. **更新应用**：检查 App Store 是否有新版本可用
5. **重新安装**：卸载后重新安装（注意：会清空所有数据，请先备份）

### Q15: 为什么 AI 润色功能无法使用？
**A:** 可能的原因：
1. **网络连接问题**：检查是否连接到互联网
2. **服务繁忙**：AI 服务提供商可能暂时繁忙，请稍后重试
3. **文本过长**：单次润色的文本不宜超过 2000 字
4. **敏感内容**：某些敏感词可能导致 API 拒绝处理

## 使用教程

### 📝 创建第一个故事

#### 步骤 1：进入编辑页面
- 在时间轴页面，点击右上角的 "+" 按钮
- 或在分类页面，点击分类卡片的"新建故事"按钮

#### 步骤 2：填写基本信息
- **标题**：输入故事标题（必填）
- **内容**：输入故事内容，支持 Markdown 格式
- **时间**：默认为当前时间，可手动调整

#### 步骤 3：添加媒体（可选）
- 点击"添加图片/视频"按钮
- 选择"拍摄"或"从相册选择"
- 可以添加多张图片或视频

#### 步骤 4：添加位置（可选）
- 点击"添加位置"按钮
- 选择"使用当前位置"或"搜索地点"
- 确认位置信息

#### 步骤 5：选择分类（可选）
- 点击"选择分类"按钮
- 浏览并选择合适的分类
- 支持多级分类导航

#### 步骤 6：保存故事
- 点击右上角的"保存"按钮
- 故事会自动保存并在时间轴上显示

5. 父分类自动选择当前二级分类

- **列表模式**：以列表形式展示，信息更紧凑

## 权限说明

### 📸 相机权限
- **用途**：拍摄照片或录制视频添加到故事中
- **申请时机**：首次点击相机按钮时
- **可否拒绝**：可以，拒绝后无法使用拍摄功能，但可以从相册选择
- **如何修改**：iPhone 设置 > 隐私与安全性 > 相机 > 我的故事

### 🖼️ 相册权限
- **用途**：选择已有的照片或视频添加到故事中
- **申请时机**：首次点击选择图片/视频按钮时
- **可否拒绝**：可以，拒绝后无法从相册选择，但可以使用拍摄功能
- **如何修改**：iPhone 设置 > 隐私与安全性 > 照片 > 我的故事

### 📍 位置权限
- **用途**：记录故事发生的地点
- **申请时机**：首次使用位置功能时
- **权限类型**：使用期间（不在后台获取位置）
- **可否拒绝**：可以，位置功能是完全可选的
- **如何修改**：iPhone 设置 > 隐私与安全性 > 定位服务 > 我的故事

### 📡 本地网络权限
- **用途**：设备间点对点数据同步
- **申请时机**：首次使用数据同步功能时
- **可否拒绝**：可以，拒绝后无法使用设备间数据迁移功能
- **如何修改**：iPhone 设置 > 隐私与安全性 > 本地网络 > 我的故事

### 🔵 蓝牙权限
- **用途**：与附近设备建立数据同步连接
- **申请时机**：首次使用数据同步功能时
- **可否拒绝**：可以，拒绝后无法使用设备间数据迁移功能（如果设备不在同一 WiFi 下）
- **如何修改**：iPhone 设置 > 隐私与安全性 > 蓝牙 > 我的故事

## 数据安全

### 🔒 数据存储安全

#### 本地存储
- **所有数据都存储在您的设备本地**
- 使用 iOS 系统的安全沙盒机制
- 其他应用无法访问您的数据

#### 数据传输安全
- **设备间同步**：点对点传输，数据在传输过程中通过本地网络传输
- **PIN 码验证**：6 位 PIN 码确保数据传输到正确的设备
- **不经过服务器**：点对点传输，数据不上传到任何互联网服务器

### 🛡️ 隐私保护

#### 我们不收集
- ❌ 您的个人身份信息
- ❌ 设备唯一标识符（用于追踪）
- ❌ 联系人列表
- ❌ 浏览历史
- ❌ 其他应用使用情况

#### 我们仅处理
- ✅ 您主动输入的故事内容（仅本地存储）
- ✅ 您选择添加的图片和视频（加密存储）
- ✅ 您授权获取的位置信息（仅本地存储）
- ✅ 您的个性化设置（语言、主题等）

#### AI 服务说明
- 使用 AI 文本润色时，您的文本会发送到**阿里云通义千问 API**
- 我们**不会**在服务器端保存您的文本内容
- AI 服务提供商可能按其隐私政策处理数据
- 建议不要在 AI 润色中输入敏感个人信息

### 🗑️ 数据删除

#### 删除单个故事
- 在时间轴页面，向左滑动故事卡片
- 点击"删除"按钮
- 确认删除
- 故事及其关联的媒体文件会被永久删除

#### 删除分类
- 在分类页面，向左滑动分类卡片
- 点击"删除"按钮
- 系统会提示将删除的内容（子分类、故事、媒体）
- 确认后永久删除

#### 清理缓存
- 进入"设置" > "缓存清理"
- 清理临时文件和压缩包
- 不会影响您的故事和媒体数据

#### 完全删除数据
- 卸载应用会删除所有本地数据
- 建议卸载前先使用"数据同步"功能导出数据

## 故障排除

### ⚠️ 应用无法启动

**可能原因**：
- iOS 版本过低
- 存储空间不足
- 应用文件损坏

**解决方案**：
1. 检查 iOS 版本是否为 16.0 或更高
2. 清理设备存储空间，至少保留 1GB 可用空间
3. 尝试重启设备
4. 重新安装应用（注意备份数据）

### ⚠️ 图片无法加载

**可能原因**：
- 文件损坏或丢失
- 存储空间不足
- 权限问题

**解决方案**：
1. 检查相册权限是否开启
2. 确保设备有足够的存储空间
3. 尝试重新添加图片
4. 如果问题持续，联系技术支持

### ⚠️ 视频无法播放

**可能原因**：
- 视频格式不支持
- 文件损坏
- 系统资源不足

**解决方案**：
1. 支持的视频格式：MP4, MOV
2. 尝试重启应用
3. 检查设备存储空间
4. 尝试重新导入视频

### ⚠️ 位置无法获取

**可能原因**：
- 位置权限未开启
- GPS 信号弱
- 定位服务关闭

**解决方案**：
1. 检查是否开启位置权限（iPhone 设置 > 隐私与安全性 > 定位服务）
2. 确保定位服务已开启
3. 移动到开阔区域，避免室内或高楼遮挡
4. 尝试手动搜索地点

### ⚠️ AI 润色失败

**可能原因**：
- 网络连接问题
- AI 服务繁忙
- 文本内容问题

**解决方案**：
1. 检查网络连接是否正常
2. 稍后重试（服务可能暂时繁忙）
3. 减少文本长度（建议不超过 2000 字）
4. 检查文本是否包含敏感词

### ⚠️ 数据同步失败

**可能原因**：
- 设备不在同一网络
- 蓝牙或 WiFi 未开启
- PIN 码输入错误

**解决方案**：
1. 确保两台设备在同一 WiFi 网络下
2. 或确保两台设备都开启了蓝牙
3. 检查 PIN 码是否输入正确
4. 关闭应用重新尝试
5. 重启两台设备

### ⚠️ 应用卡顿或缓慢

**可能原因**：
- 设备性能不足
- 存储空间不足
- 缓存文件过多
- 数据量过大

**解决方案**：
1. 清理设备存储空间
2. 使用"缓存清理"功能
3. 关闭后台其他应用
4. 重启设备
5. 如果数据量很大（超过 1000 个故事），考虑归档旧数据

### ⚠️ 界面显示异常

**可能原因**：
- 系统语言设置问题
- 主题冲突
- 应用缓存问题

**解决方案**：
1. 尝试切换语言后再切换回来
2. 重新选择主题
3. 重启应用
4. 重启设备
